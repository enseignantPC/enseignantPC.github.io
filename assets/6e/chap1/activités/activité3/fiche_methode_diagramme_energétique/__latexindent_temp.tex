\newcommand{\introduction}{
    Tout changement dans le monde qui nous entoure peut être compris 
    à travers la notion d'énergie.
    Lors d'une \emph{conversion énergétique}, l'énergie peut passer 
    d'une forme à une autre.
    Par exemple, la lumière du soleil qui 
    rayonne sur nous est de l'énergie \emph{lumineuse}, lorsque
    cette lumière réchauffe notre peau, 
    l'énergie est sous forme \emph{thermique}: 
    Il y a eu changement de type d'énergie, c'est une \textbf{conversion d'énergie.}
    
    \vspace{9pt}
    Pour comprendre le monde en terme d'énergie, il faut maîtriser deux techniques:
    \begin{itemize}
        \item le \textbf{diagramme de conversion d'énergie} d'un objet.
        \item le \textbf{diagramme d'énergie} d'une situation (qui contient plusieurs objets).
    \end{itemize}

    \vspace{-5pt}
    }

    \tcbsubtitle{Diagramme de conversion d'énergie}

        Le \textbf{diagramme de conversion d'énergie} d'un objet indique :
        \begin{itemize}
            \item Quel type d'énergie est nécessaire pour utiliser l'objet.
            \item Quel type d'énergie l'objet va émettre quand il est utilisé.
        \end{itemize}

        \vspace{4pt}
        \underline{Exemple} : un sèche cheveux utilise de \textit{l'énergie électrique}
        pour projeter de l'air chaud en mouvement. l'air en mouvement correspond 
        à de \textit{l'énergie mécanique}, la chaleur de l'air correspond à de \textit{l'énergie thermique}.
            \begin{center}
                \tcbincludegraphics[width=15cm,title=diagramme de conversion d'un séche cheveux,colback=white]{diagramme_seche_cheveux.png}
            \end{center}

        \begin{itemize}
            \item Pour un diagramme de conversion :
                \begin{enumerate}
                    \item Il faut écrire le nom de l'objet considéré dans un cercle.
                    \item Faire une flèche à gauche avec le nom de l'énergie que l'objet utilise.
                    \item Faire une ou plusieurs flèches à droite avec 
                    le nom de l'énergie que l'objet donne.
                \end{enumerate} 

            \item Pour un diagramme d'énergie:
                \begin{enumerate}
                    \item On commence tout à gauche avec un objet ou un type d'énergie.
                    \item On rajoute tout les objets à la suite 
                    en utilisant la méthode vue pour un diagramme de conversion.
                \end{enumerate}
        \end{itemize}
